Principais Formatações

Negrito | Destaque

Nesta frase temos um termo em negrito usando a tag <b>(não semântica)

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico | Ênfase

Nesta frase temos um termo em itálico usando a tag <i>(não semântica)

Nesta frase temos um termo em ênfase usando a tag <em>(semântica)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

E aqui temos outro texto marcado com cor diferente usando a mesma tag <mark>

Texto Grande e Pequeno

Temos um texto grande e um texto pequeno usando as tags <big> e <small>

Texto deletado

Temos um texto excluído usando a tag <del>

Texto inserido

Temos um texto inserido usando a tag <ins>

Também temos um texto inserido usando a tag <u>

Texto sobrescrito

Para inserir fórmulas matemáticas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir fórmulas químicas do tipo H20 usando a tag <sub>
